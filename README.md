# Object-Oriented Calculator

This is a simple calculator built using Object-Oriented Programming (OOP) principles in JavaScript.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Built with JavaScript classes and methods
- Manages state and events in an OOP style

## How to Run
Simply open the HTML file in your browser, and you can start using the calculator right away.
